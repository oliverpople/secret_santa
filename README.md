## User Stories

````
As a user
So I can be part of secret santa
I want to add my name to the list

As a user
So I no one knows who my secret is
I want my secret santa to be randomly selected

As a user
So I don't have to give myself a gift
I want my secret santa to be anyone in the group but me

As a user
So I can give my gift to the right person
I want to find out who I'm secret santa for
````
